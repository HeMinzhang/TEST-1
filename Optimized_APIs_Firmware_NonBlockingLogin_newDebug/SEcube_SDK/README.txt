SEcube(tm) Open Source SDK

This is the SDK of SEcube(tm).
It is composed by:
SEcube(tm), SEfile(tm), SElink(tm).


The folders:
-"Development"
    Contains all the resources needed to employ the SEcube(tm) Open Source Libraries
	
-"Documentation"
    Contains the technical documentation for SEcube(tm) Open Source Libraries: SEcube(tm), SEfile(tm), SElink(tm)
	
-"Libraries"
    Contains the source code of the SEcube(tm) Open Source Libraries: SEcube(tm), SEfile(tm), SElink(tm)
